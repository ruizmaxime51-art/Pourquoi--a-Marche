# v24 — nouveaux liens affiliés

Produits ajoutés au registre centralisé :

- pH-mètre numérique ;
- étiquettes autocollantes ;
- lunettes de sécurité ;
- gants chimiques réutilisables ;
- gants jetables en nitrile.

Principes éditoriaux conservés :

- les gants jetables ne sont pas présentés comme protection principale contre la soude concentrée ;
- la compatibilité des gants doit être vérifiée selon la documentation du fabricant ;
- le pH-mètre exige étalonnage, entretien et stockage correct de l’électrode ;
- les liens utilisent `rel="nofollow sponsored noopener"` via le moteur centralisé.
